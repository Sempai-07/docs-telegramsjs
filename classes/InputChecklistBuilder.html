Class InputChecklistBuilder

Builder for creating Telegram checklist inputs

Example

const checklist = new InputChecklistBuilder()
  .setTitle('My Checklist')
  .addTask('Task 1')
  .addTask('Task 2')
  .setOthersCanAddTasks(true);
